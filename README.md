Это Android-приложение построено с учетом современных практик разработки и демонстрирует следующие возможности:

<b>Проектирование</b>
<ul>
<li>Полный пользовательский интерфейс, разработанный с использованием Jetpack Compose</li>

<li>UI реализован на основе дизайна макета Figma</li>

<li>Имеет пользовательские визуальные компоненты</li>
</ul>

<b>Архитектура и паттерны</b>
<ul>
<li>Чистая архитектура с четким разделением слоев</li>

<li>Слой представления MVVM (Model-View-ViewModel)</li>

<li>MVI (Model-View-Intent) для однонаправленного потока данных</li>

<li>Многомодульная структура для лучшей масштабируемости и разделения</li>
</ul>


<b>Технический стек</b>
<ul>
<li>Внедрение зависимостей с помощью Hilt</li>

<li>Получение данных сети с помощью REST API (Retrofit)</li>

<li>Сохранение данных в локальную базу (Room)</li>

<li>Coil для эффективной загрузки изображений</li>

<li>Kotlin Coroutines для фоновых операций</li>

<li>Flow для реактивных потоков данных</li>

Пользовательская реализация эффекта размытия без сторонних библиотек</li>
</ul>


<b>Особенности</b>
<ul>
<li>При каждой генерации карточки изображение подбирается случайным образом, в базу информация по картинкам не пишется</li>

<li>Кастомный компонент GlassyBox с эффектом матового стекла.</li>

<li>Глубокая оптимизация производительности: использование Deferred State Reading (отложенного чтения состояния) и лямбда-модификаторов для исключения лишних рекомпозиций.</li>

<li>Применение SubcomposeLayout для точного измерения динамических размеров компонентов в реальном времени.</li>
</ul>
