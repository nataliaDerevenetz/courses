Это Android-приложение построено с учетом современных практик разработки и демонстрирует следующие возможности:

<b>Проектирование</b>

Полный пользовательский интерфейс, разработанный с использованием Jetpack Compose

UI реализован на основе дизайна макета Figma

Имеет пользовательские визуальные компоненты


<b>Архитектура и паттерны</b>

Чистая архитектура с четким разделением слоев

Слой представления MVVM (Model-View-ViewModel)

MVI (Model-View-Intent) для однонаправленного потока данных

Многомодульная структура для лучшей масштабируемости и разделения


<b>Технический стек</b>

Внедрение зависимостей с помощью Hilt

Получение данных сети с помощью REST API (Retrofit)

Сохранение данных в локальную базу (Room)

Coil для эффективной загрузки изображений

Асинхронные операции

Kotlin Coroutines для фоновых операций

Flow для реактивных потоков данных

Пользовательская реализация эффекта размытия без сторонних библиотек


<b>Особенности</b>

При каждой генерации карточки изображение подбирается случайным образом, в базу информация по картинкам не пишется

Продвинутый Jetpack Compose:

Кастомный компонент GlassyBox с эффектом матового стекла.

Глубокая оптимизация производительности: использование Deferred State Reading (отложенного чтения состояния) и лямбда-модификаторов для исключения лишних рекомпозиций.

Применение SubcomposeLayout для точного измерения динамических размеров компонентов в реальном времени.
